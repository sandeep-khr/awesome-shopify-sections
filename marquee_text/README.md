# Infinite Marquee

Create hype and urgency with this smooth scrolling text banner. Perfect for flash sales, announcements, or displaying brand values.

![Preview](preview.png)

## Features
- **Seamless Loop**: Pure CSS animation for silky smooth scrolling.
- **Customizable**: Control speed, direction (Left/Right), and pausing behavior.
- **Content Blocks**: Add unlimited text blocks with optional links.
- **No JS Bloat**: Runs without heavy JavaScript libraries.

## Installation

1. **Copy Code**: Open `section-marquee.liquid` and copy the contents.
2. **Add Section**:
   - Shopify Admin > Online Store > Themes > Edit Code.
   - Sections > "Add a new section".
   - Name it `marquee-text`.
   - Paste code.

## Usage

1. **Add to Template**: In Theme Editor, add "Infinite Marquee".
2. **Add Content**: Click "Add Announcement" to add text blocks.
3. **Settings**:
   - **Speed**: Adjust animation duration (lower is faster).
   - **Direction**: Choose Left-to-Right or Right-to-Left.
   - **Pause on Hover**: Toggle this to let users read text easily.
